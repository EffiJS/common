```
npx effijs init
```
Will create project structure
```
src
    assets
    components
    modules
        App
            app.controller.ts
            app.navigation.ts
            app.service.ts
            index.ts
    index.ts
```

```
npx effijs -m User
```
Will create in 'modules' User module

```
    ...
    modules
        ...
        User
            Componets
                List.tsx
            Screens
                List.tsx
                
            user.controller.ts
            user.service.ts
            user.navigation.ts
            index.ts
```
